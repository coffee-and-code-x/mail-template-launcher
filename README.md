# Mail Template Launcher

Outlook のメール作成画面をワンクリックで開く Excel VBA ツール。  
テンプレートに登録した宛先・CC・件名・本文を自動入力し、`.Display` で表示するので確認してから手動送信できる。

---

## 必要環境

- Windows
- Microsoft Excel（マクロ有効）
- Microsoft Outlook（インストール済み）
- PowerShell 5.1 以上（ビルド時のみ）

---

## セットアップ

### 1. ビルド（.xlsm を生成する）

Excel の「VBA プロジェクト オブジェクト モデルへのアクセスを信頼する」を有効にしてから実行する。

```
Excel → ファイル → オプション → トラスト センター → トラスト センターの設定
→ マクロの設定 → 「VBA プロジェクト オブジェクト モデルへのアクセスを信頼する」にチェック
```

PowerShell でビルドスクリプトを実行する。

```powershell
powershell -ExecutionPolicy Bypass -File .\build.ps1
```

`mail_template_launcher.xlsm` が生成される。

### 2. シートの初期化（初回のみ）

1. `mail_template_launcher.xlsm` を開く
2. `Alt + F8` → `SetupSheets` を選択 → 実行
3. 「テンプレート一覧」シートと「本文_1〜5」シートが自動作成される

---

## シート構成

### テンプレート一覧

| セル | 内容 |
|------|------|
| A2〜A4 | 項目のラベル名（編集可。プレースホルダー名になる） |
| B2〜B4 | 案件情報の入力値（メール送信のたびに入力する） |
| B5 | Outlook 実行ファイルのパス（任意。Office 365 と 2016 が混在する場合に設定） |
| 8〜12行目 | テンプレート一覧（宛先・CC・件名・本文シート名・起動ボタン） |

### 本文_1〜5

各テンプレートの本文を `A2` セルに記述する。  
右上のボタンをクリックするとプレースホルダーがクリップボードにコピーされ、`Ctrl+V` で貼り付けられる。

---

## 使い方

### テンプレートの設定（初回）

1. `テンプレート一覧` の A2〜A4 にラベル名を入力する（例：`顧客名:` `案件名:` `担当者:`）
2. 8〜12行目の宛先・CC・件名を入力する
3. 対応する `本文_N` シートの A2 に本文を入力する
4. C列で形式を選択する（`HTML` または `TEXT`）

### メール送信（毎回）

1. `テンプレート一覧` の B2〜B4 に案件情報を入力する
2. 送信したいテンプレートの「起動」ボタンをクリックする
3. Outlook のメール作成画面が開く
4. 内容を確認して手動で送信する

---

## プレースホルダー

A列のラベル名（末尾の `:` は除去）が自動的にプレースホルダー名になる。

| A列のラベル | 使えるプレースホルダー |
|-----------|-------------------|
| `顧客名:` | `{顧客名}` |
| `案件名:` | `{案件名}` |
| `担当者:` | `{担当者}` |

プレースホルダーは宛先・CC・件名・本文のすべてに使用できる。

---

## Office 365 Outlook の設定

2016 と 365 が共存する環境では、B5 セルに 365 の実行ファイルパスを指定する。

```
C:\Program Files\Microsoft Office\root\Office16\OUTLOOK.EXE
```

## ファイル構成

```
mail-template-launcher/
├── build.ps1               ビルドスクリプト
└── src/vba/
    ├── Module_Template.bas テンプレート読み取り・プレースホルダー置換
    ├── Module_Outlook.bas  Outlook COM 連携
    ├── Module_Launcher.bas 起動ボタンのエントリポイント（Launch_1〜5）
    ├── Module_Setup.bas    初回セットアップマクロ
    └── ThisWorkbook.cls    Workbook_Open イベント
```
